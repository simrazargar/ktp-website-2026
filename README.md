# Kappa Theta Pi - USC Website

A modern, responsive website for Kappa Theta Pi, USC's premier professional technology fraternity.

## Features

- **Modern Design**: Clean, professional dark theme matching the fraternity's brand
- **Responsive Layout**: Fully responsive design that works on all devices
- **Sections**:
  - Hero section with call-to-action buttons
  - Who We Are section
  - Our Values (6 value propositions)
  - Network and Alumni Companies
  - Social Life showcase
  - Contact form with social media links

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Tech Stack

- **React 18**: Modern React with hooks
- **Vite**: Fast build tool and dev server
- **CSS3**: Custom styling with modern CSS features
